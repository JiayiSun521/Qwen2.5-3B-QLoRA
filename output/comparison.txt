问题：写一个 Python 冒泡排序函数

微调前：
当然！冒泡排序是一种简单的排序算法，它重复地遍历要排序的列表，依次比较相邻的元素。如果它们的顺序错误就把它们交换过来。遍历列表的工作是重复进行的，直到没有再需要交换的位置为止。

下面是一个用 Python 实现的冒泡排序函数：

```python
def bubble_sort(arr):
    n = len(arr)
    # 遍历所有数组元素
    for i in range(n):
        # 最后 i 个元素已经是排好序的
        for j in range(0, n-i-1):
            # 如果当前元素大于下一个元素，则交换它们
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# 测试代码
arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = bubble_sort(arr)
print("排序后的数组:", sorted_arr)
```

在这个实现中：
1. 外层循环 `for i in range(n)` 控制遍历的次数。
2. 内层循环 `for j in range(0, n-i-1)` 比较相邻元素并进行交换。
3. 如果当前元素大于下一个元素，则交换它们的位置。
